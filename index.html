Web paint

Стек: React.js + Node.js + Websocket + MobX + Canvas + Sass

Клиент-серверное приложение для совместного рисования на одном холсте. Работает в режиме реального времени, изменения которые внес один пользователь тут же будут отображаться у всех остальных подключенных к данной сессии. Есть возможность сохранять получившееся изображение к себе на компьютер.

Fake blog

Стек: React.js + Css

Данное приложение использует стороннее api с jsonplaceholder.typicode.com имитируя постраничную подгрузку постов в блог. Реализованы возможности поиска по записям, их сортировка, добавление своего нового поста.

Coin shop

Стек: React.js + Node.js + Express + PostgeSQL + MobX + Bootstrap

Интернет магазин монет. Присутствует регистрация и авторизация по jwt токену. При входе с ролью админа открывается возможность добавлять новую страну и продукты. Каждую монету можно рассмотреть получше на ее отдельной странице.

Todo list

Стек: React.js + Scss + Node.js + PostgeSQL

Лист задач с собсвтенным Rest Api. Можно создавать, редактировать, удалять задачу, а так же помечать как выполненную. Если срок выполнения истек, будет отображаться соответствующее оповещение. Доступно неограниченное прикрепление файлов к задаче.
